# OmniAgent AI – Autonomous Multi-Agent System with Tool Execution

An advanced **multi-agent AI system** that can plan, execute, and perform real-world tasks using tools like GitHub, APIs, file handling, and web browsing.

---

## 🚀 Features

- 🧠 Multi-Agent Architecture (Planner, Executor, Tool Agent)
- 🛠 Multi-Tool Execution
  - GitHub Repo Creation
  - File Writing
  - API Fetching
  - Web Browsing
- 🔄 Dynamic Tool Selection using LLM
- 📊 Streamlit Dashboard UI
- 💾 Output Persistence
- ⚡ Built with LangGraph + FastAPI + Ollama

---

## 🧠 Architecture

User Input → Planner → Execution → Tool Agent → Real Actions

---

## 🛠 Tech Stack

- Python
- LangGraph
- FastAPI
- Streamlit
- Ollama (LLM)
- GitHub API
